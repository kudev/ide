Chart 1

The motivation for this chart was to see how the temperature has changed over the past 140 years in Copenhagen. Claims of global warming are certainly widespread enough to warrant some investigation, and this chart aims to check if there seems to be any effect of the global warming in the Danish capital.

The data set was pulled from temperature measurements in Copenhagen.

Since 1880, the average temperature has risen by around 1.5 degrees celsius per year, as is displayed by the trend line.
It is also evident that the year-to-year average temperature has many spikes, as it can vary by several degrees.
Lastly, in the later years of the graph the records for highest average temperature has been set.

The gathered results suggests that there is a tendency for the average yearly temperature to increase, although on a year-by-year basis, we can have the same average temperature for a year as a hundred years ago, i.e. what was called a warm year a hundred years ago would be called cold nowadays.

Our implementation is relatively straightforward, it follows the same patterns as most resources we found online. The trend line implementation was mainly borrowed from a website (credited in the source), but we did modify it to suit our needs. We were also experimenting with the various line interpolation techniques to arrive at the current, soothingly curving one (e.g. the linear option was very 'spiky').

Chart 2

The motivation for the second graph was to check if there is a correlation in temperatures between Egedesminde in Greenland, and Copenhagen, that is, we were curious if the global warming affected the two areas in the same way or not.

The data set was taken from the same website for Copenhagen and Egedesminde.

If there was a strong correlation between the two temperatures, there would be a roughly diagonal line from the bottom left corner (cold in Egedesminde and Copenhagen) to the top right corner (hot in Egedesminde and Copenhagen). Instead we get an almost cubic scattering of the data points.
There are notable outliers in the data, and the biggest difference in temperatures in Egedesminde occur at almost the same temperature in Copenhagen.

The gathered results show that the temperature shifts in Egedesminde and Copenhagen are are not strongly correlated, which shows that the world is not uniformly affected by global warming. This is, of course, not necessarily surprising given that Copenhagen is a capital and probably generates a lot more carbon-dioxide, among other factors.

The scatterplot implementation was mainly written by us based on the scatterplot example in the d3 gallery. We noticed that when using only the needed ranges for the axes, some of the points ended up on the axes themselves, so we extended the ranges by one in all directions.
